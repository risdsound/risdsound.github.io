foafx released

July 17, 2022

Version 1 of foafx has been released.   

foafx is a command line tool for applying spatially positioned audio effects to first order ambisonic sound files. It is written with Elementary, a JavaScript framework for writing audio applications, and uses its offline-renderer package.

During the Spring 2022 semester, our team worked with Nick Thompson from Elementary Audio to explore and prototype software tools for spatial audio.

The source code and additional information is available on GitHub at https://github.com/risdsound/foafx. The tool itself is available on the command line from npm.

Install from your terminal and read the help files with the below commands:

npm install -g foafx
foafx --help
